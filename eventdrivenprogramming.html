What are the different styles of programming

There are many types of programming styles, four of which are Sequential
programming, Object Oriented programming, Event driven programming and Concurrent
programming. Each has benifits and draw backs, here we'll talk about Event Driven
Programming.

Event Driven Programming Explanation

Event driven programming is a stype of programming, where the program is designed to
respond to events.

These events can be interations with a user, or with the environment, for example a
button being pressed, entry into a text box or a sensor detecting the outside
temperature.

Each event will have an appropriate response, this response is linked to the event by
an event handler which performs an action by triggering a call back function.

As an event can happen at any time, the event handler is 'active' within a loop and so
i always 'listening'

In contrast to sequential programming, event driven programming is
non-deterministic. The control and flow of a program can is decided by events which
trigger functions, but the timing of each function has infinite possibly permutations
and the possible paths of even a small program may be imposiible to map out
completely.

the fact that it is non-deterministic leads to uncertainty so rather than
understanding a program in the way we can understand a sequential program,
understanding what might happrn at what point in time, we can understand a program by
determining the state of the program.

If we were to look at a simple program which uses an event driven program to turn on
a lightbulb when a switch is pressed. The state of the program before the switch is
pressed is a light that is turned off. The state after the switch being pressed is
that the light is turned on.

The number of states however can get very large very quickly. If there were for
example 20 switches for 31 different devices, all with two states, on and off, we
have 2 to the power of 31 possible states that the program could find itself in,
being 2,147,483,648 which is more than 2 billion.
